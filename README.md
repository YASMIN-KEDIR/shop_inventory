# 🛒 Shop Inventory & Sales Ledger

A professional Laravel-based inventory management system designed for electronics shops to track stock, calculate real-time profit, and manage daily sales.

## 🚀 Features
* **Real-time Profit Tracking:** Calculates projected profit and actual money earned.
* **Inventory Management:** Track buying/selling prices and stock levels.
* **Low Stock Alerts:** Automatic visual cues when stock is running low.
* **Search & Filter:** Quickly find items among large inventories.
* **Potential Income Overview:** See the total value of your shop at a glance.

## 🛠️ Tech Stack
* **Backend:** Laravel 11 / PHP 8.2
* **Database:** MySQL
* **Frontend:** Blade Templates & CSS (Custom UI)
